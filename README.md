RPCI All Markets Study
This repository presents the implementation of the Relative Performance Convergence Index (RPCI) to analyse dynamic convergence and herding behaviour across global equity markets.
Overview
This study evaluates herding behaviour using both traditional dispersion-based models and a performance-based framework.
•Models used: CSAD, CSSD, RPCI
•Markets analysed: FTSE 100, S&P 500, DAX 40, NIFTY 50, China
•Period: 2018–2026
Traditional models (CSAD, CSSD) show weak or insignificant evidence of herding, while RPCI captures statistically significant convergence during crisis periods, highlighting the dynamic nature of investor behaviour. [1]1
Key Contribution
•Introduces and applies the RPCI framework as a dynamic, performance-driven measure
•Demonstrates limitations of dispersion-based herding measures
•Provides cross-market empirical evidence of time-varying convergence behaviour
